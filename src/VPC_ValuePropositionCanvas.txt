################################################################################
                    QUE ES EL VALUE PROPOSITION CANVAS?
################################################################################

El VPC es una herramienta de estrategia creada por Alexander Osterwalder que
ayuda a disenar productos que realmente resuelvan necesidades del cliente.

Se divide en DOS partes que deben "encajar" (FIT):

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ┌───────────────────────┐              ┌───────────────────────┐         │
│   │    MAPA DE VALOR      │    ←FIT→     │   PERFIL DEL CLIENTE  │         │
│   │    (Value Map)        │              │   (Customer Profile)  │         │
│   │                       │              │                       │         │
│   │ • Productos/Servicios │              │ • Jobs (Trabajos)     │         │
│   │ • Pain Relievers      │              │ • Pains (Dolores)     │         │
│   │ • Gain Creators       │              │ • Gains (Ganancias)   │         │
│   └───────────────────────┘              └───────────────────────┘         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

PERFIL DEL CLIENTE:
- Jobs: Que tareas intenta completar el cliente?
- Pains: Que le frustra o le impide lograrlo?
- Gains: Que resultados o beneficios desea obtener?

MAPA DE VALOR:
- Productos/Servicios: Lo que ofreces
- Pain Relievers: Como alivias los dolores del cliente
- Gain Creators: Como generas los beneficios que busca
